# Spider ML Task 1

This repository contains my submissions for Spider ML Task 1.

## Base Task

Implemented a Fashion-MNIST image classification model using PyTorch.

Files included:

* Training notebook
* Saved model weights (`model.pkl`)
* Test predictions (`submission.csv`)
* Accuracy and loss plots

## Applied ML Domain – RAG

Built a Retrieval-Augmented Generation (RAG) system that can answer questions from multiple research papers.

Libraries used:

* Google Gemini
* ChromaDB
* Sentence Transformers
* Streamlit
* PyPDF

Features:

* Multi-PDF ingestion
* Semantic search and retrieval
* Question answering using retrieved context
* Source paper citation
